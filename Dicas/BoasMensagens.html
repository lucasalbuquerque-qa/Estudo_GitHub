Boas mensagens de commit Dicas

Uma boa mensagem de commit explica o que foi feito e por quê, de forma clara e objetiva. Isso torna o histórico do projeto fácil de entender semanas ou meses depois.

A regra de ouro

Escreva a mensagem como se fosse completar a frase: "Se aplicado, este commit vai..."

# Bom
git commit -m "Adiciona validação no formulário de login"
git commit -m "Corrige erro 404 na rota de perfil"
git commit -m "Remove dependência não utilizada"

# Ruim
git commit -m "fix"
git commit -m "alterações"
git commit -m "atualizando coisas"

Use o modo imperativo

Escreva no imperativo: Adiciona, Corrige, Remove, Atualiza — não "Adicionado", "Corrigindo" ou "Adicionei".

# Imperativo (correto)
"Adiciona página de contato"
"Corrige bug no carrinho"
"Atualiza README com instruções"

# Outros modos (evite)
"Adicionando página de contato"
"Bug do carrinho corrigido"
"Adicionei o README"

Seja específico e curto

Limite o título a 72 caracteres. Se precisar de mais detalhes, adicione uma descrição após uma linha em branco:

git commit -m "Corrige validação de e-mail no cadastro

O campo aceitava e-mails sem o domínio (.com, .br etc).
Adicionada regex de validação antes de enviar o formulário."

Agrupe mudanças relacionadas

Cada commit deve fazer uma coisa só. Não misture correção de bug com nova funcionalidade no mesmo commit:

# Commits separados e claros
git commit -m "Corrige alinhamento do header no mobile"
git commit -m "Adiciona botão de compartilhar no post"

# Commit que mistura tudo (evite)
git commit -m "Várias correções e melhorias"

Referenciar issues do GitHub

Se o commit resolve um issue, mencione o número para vincular automaticamente:

git commit -m "Corrige bug no login — fecha #42"
git commit -m "Implementa pesquisa — resolve #87"

Exemplo completo

# Ruim
git commit -m "coisas"

# Bom
git commit -m "Adiciona máscara de CPF no formulário de cadastro"

# Ótimo (com descrição)
git commit -m "Adiciona máscara de CPF no formulário de cadastro

Usuários relatavam confusão ao preencher o campo.
A máscara formata automaticamente no padrão 000.000.000-00."
💡 Dica: Pense no seu eu do futuro lendo o histórico às 23h tentando entender o que aconteceu. Uma boa mensagem de commit é um presente para você mesmo.
