git stash Desfazer / Corrigir

← Voltar

O comando git stash guarda temporariamente as alterações que ainda não foram commitadas, deixando o projeto limpo para você trocar de branch ou fazer outra tarefa.

Quando usar

Use git stash quando precisar trocar de branch mas ainda não quer fazer um commit com as alterações atuais.

Guardar as alterações

Salva as alterações atuais no stash e limpa o projeto:

git stash

Guardar com uma descrição

Adiciona uma mensagem para identificar o que foi guardado:

git stash save "descrição do que estava fazendo"

Ver o que está guardado

Lista todos os stashes salvos:

git stash list

Recuperar as alterações

Traz de volta as alterações guardadas:

git stash pop

Exemplo completo

git stash save "ajuste no formulário"
git checkout main
git checkout minha-branch
git stash pop
💡 Dica: git stash pop aplica e remove do stash. Se quiser manter no stash após aplicar, use git stash apply.
