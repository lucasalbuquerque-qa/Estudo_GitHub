git add Salvar Alterações

O comando git add prepara os arquivos para o commit, adicionando-os à área de stage. Só os arquivos no stage serão incluídos no próximo commit.

Quando usar

Use git add depois de modificar ou criar arquivos, para marcá-los como prontos para serem salvos no histórico.

Adicionar um arquivo específico

Adiciona apenas um arquivo ao stage:

git add nome-do-arquivo.html

Adicionar todos os arquivos

Adiciona todos os arquivos modificados e novos de uma vez:

git add .

Verificar o que foi adicionado

Confirme quais arquivos estão no stage:

git status

Exemplo completo

git status
git add .
git status
💡 Dica: Prefira git add . para adicionar tudo, mas use git add arquivo quando quiser separar alterações em commits diferentes.
