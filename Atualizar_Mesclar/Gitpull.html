git pull Atualizar / Mesclar

← Voltar

O comando git pull baixa as alterações mais recentes do repositório remoto e as aplica automaticamente no projeto local. É uma combinação de git fetch + git merge.

Quando usar

Use git pull antes de começar a trabalhar, para garantir que seu projeto local está atualizado com o que está no GitHub.

Verificar a branch atual

Confirme em qual branch você está:

git branch

Atualizar o projeto

Baixa e aplica as alterações remotas:

git pull

Atualizar de uma branch específica

Indica o repositório remoto e a branch de origem:

git pull origin main

Verificar após atualizar

Confirme que o projeto está em dia:

git status

Exemplo completo

git branch
git pull origin main
git status
💡 Dica: Sempre faça git pull antes de começar a editar arquivos em projetos colaborativos para evitar conflitos desnecessários.
