git switch Branch (Ramos)

O comando git switch é a forma moderna de trocar de branch, introduzido no Git 2.23 como substituto mais claro do git checkout. Faz apenas uma coisa: mudar de branch.

Quando usar

Use git switch sempre que precisar mudar de branch. É mais simples e direto que o git checkout, que acumula várias funções diferentes.

Mudar para outra branch

Troca para uma branch que já existe:

git switch nome-da-branch

Criar e já entrar na branch

Cria uma nova branch e muda para ela com o -c:

git switch -c nova-branch

Voltar para a branch anterior

O traço - volta para a última branch que você estava:

git switch -

Voltar para a branch main

Retorna para a branch principal do projeto:

git switch main

Diferença com git checkout

Enquanto o git checkout serve para branches e arquivos, o git switch é exclusivo para branches e o git restore para arquivos:

# Antigo (checkout faz tudo)
git checkout minha-branch
git checkout -b nova-branch

# Moderno (cada comando tem um papel)
git switch minha-branch
git switch -c nova-branch

Exemplo completo

git branch
git switch -c minha-feature
git branch
git switch main
git switch -
💡 Dica: O git switch só está disponível no Git 2.23 ou superior. Rode git --version para verificar se seu Git já tem suporte.
