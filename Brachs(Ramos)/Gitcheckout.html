git checkout Branch (Ramos)

← Voltar

O comando git checkout permite mudar de branch ou restaurar arquivos para um estado anterior. É usado para navegar entre diferentes versões do projeto.

Quando usar

Use git checkout para trocar de branch, criar e já entrar em uma nova branch, ou restaurar um arquivo específico.

Mudar para outra branch

Troca para uma branch que já existe:

git checkout nome-da-branch

Criar e entrar numa nova branch

Cria a branch e já muda para ela com o -b:

git checkout -b nova-branch

Voltar para a branch main

Retorna para a branch principal do projeto:

git checkout main

Restaurar um arquivo modificado

Desfaz as alterações de um arquivo específico:

git checkout -- nome-do-arquivo.html

Exemplo completo

git checkout -b minha-feature
git branch
git checkout main
💡 Dica: Em versões mais recentes do Git, git switch nome-da-branch é o comando moderno para trocar de branch.
