git branch Branch (Ramos)

← Voltar

O comando git branch gerencia as branches do repositório. Branches são ramificações independentes do projeto, usadas para desenvolver funcionalidades sem afetar o código principal.

Quando usar

Use git branch para listar, criar ou deletar branches no seu repositório.

Listar todas as branches

Mostra todas as branches locais (a atual fica marcada com *):

git branch

Criar uma nova branch

Cria uma nova branch mas não muda para ela:

git branch nome-da-branch

Deletar uma branch

Remove uma branch que já foi mergeada:

git branch -d nome-da-branch

Renomear a branch atual

Muda o nome da branch em que você está:

git branch -M novo-nome

Exemplo completo

git branch
git branch minha-feature
git branch
git branch -d minha-feature
💡 Dica: Para criar uma branch e já mudar para ela ao mesmo tempo, use git checkout -b nome-da-branch.
