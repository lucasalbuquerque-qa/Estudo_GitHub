git shortlog Utilitários

O comando git shortlog exibe um resumo do histórico de commits agrupado por autor. Mostra quantos commits cada pessoa fez e quais foram as mensagens.

Quando usar

Use git shortlog para ver quem mais contribuiu no projeto, gerar um resumo de atividade da equipe ou montar um changelog de versão.

Ver resumo por autor

Lista todos os autores com seus commits:

git shortlog

Ordenar por quantidade de commits

Mostra quem mais commitou, do mais ativo para o menos:

git shortlog -sn

Ver todos os branches

Inclui commits de todas as branches no resumo:

git shortlog -sn --all

Ver num intervalo de datas

Filtra os commits de um período específico:

git shortlog --after="2024-01-01" --before="2024-12-31"

Exemplo completo

git shortlog
git shortlog -sn
git shortlog -sn --all
💡 Dica: O git shortlog -sn é muito usado para gerar a seção de Contributors em projetos open source no README.
