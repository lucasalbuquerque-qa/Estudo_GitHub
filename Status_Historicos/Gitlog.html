git log Status e Histórico

← Voltar

O comando git log exibe o histórico de commits do repositório, mostrando quem fez cada alteração, quando e com qual mensagem.

Quando usar

Use git log para ver o histórico de mudanças do projeto, encontrar um commit específico ou entender o que foi feito ao longo do tempo.

Ver o histórico completo

Mostra todos os commits com detalhes:

git log

Histórico resumido

Mostra cada commit em uma linha:

git log --oneline

Histórico com gráfico de branches

Visualiza as branches e merges de forma gráfica:

git log --oneline --graph --decorate

Ver os últimos N commits

Para ver apenas os últimos 5 commits por exemplo:

git log -5 --oneline

Filtrar por autor

Mostra commits feitos por uma pessoa específica:

git log --author="nome"

Filtrar por período (data)

Exibe commits dentro de um intervalo de datas:

git log --since="2024-01-01" --until="2024-01-31"

Buscar por palavra na mensagem

Procura commits por termos como "bug", "fix" ou "hotfix":

git log --grep="bug"

Histórico de um arquivo específico

Mostra apenas commits que alteraram um arquivo:

git log nome-do-arquivo.js

Ver alterações detalhadas (diff)

Exibe exatamente o que foi modificado em cada commit:

git log -p

Exemplo completo

git log --oneline
git log --oneline --graph --decorate
git log -5 --oneline
💡 Dica: Pressione q para sair do log quando ele abrir no modo de navegação do terminal.
