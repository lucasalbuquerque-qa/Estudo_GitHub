git log Status e Histórico

← Voltar

O comando git log exibe o histórico de commits do repositório, mostrando quem fez cada alteração, quando e com qual mensagem.

Quando usar

Use git log para ver o histórico de mudanças do projeto, encontrar um commit específico ou entender o que foi feito ao longo do tempo.

Ver o histórico completo

Mostra todos os commits com detalhes:

git log

Histórico resumido

Mostra cada commit em uma linha:

git log --oneline

Histórico com gráfico de branches

Visualiza as branches e merges de forma gráfica:

git log --oneline --graph --decorate

Ver os últimos N commits

Para ver apenas os últimos 5 commits por exemplo:

git log -5 --oneline

Exemplo completo

git log --oneline
git log --oneline --graph --decorate
git log -5 --oneline
💡 Dica: Pressione q para sair do log quando ele abrir no modo de navegação do terminal.
