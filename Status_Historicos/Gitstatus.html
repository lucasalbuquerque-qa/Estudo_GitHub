git status Status e Histórico

O comando git status mostra o estado atual do repositório: quais arquivos foram modificados, quais estão prontos para commit e quais ainda não estão sendo rastreados.

Quando usar

Use git status sempre que quiser saber o que mudou no projeto antes de fazer um commit.

Ver o status atual

No terminal do VS Code execute:

git status

Entendendo o resultado

O Git mostra os arquivos em três categorias:

# Arquivos prontos para commit (em verde)
Changes to be committed

# Arquivos modificados mas não adicionados (em vermelho)
Changes not staged for commit

# Arquivos novos sem rastreamento
Untracked files

Versão resumida

Para ver um resumo mais compacto:

git status -s

Exemplo completo

git status
git status -s
💡 Dica: Sempre rode git status antes de fazer um git add ou git commit para não enviar arquivos errados.
