git diff Status e Histórico

← Voltar

O comando git diff mostra exatamente o que foi alterado nos arquivos, linha por linha, antes de fazer o commit.

Quando usar

Use git diff para revisar as mudanças feitas nos arquivos antes de adicioná-los ou commitá-los.

Ver todas as alterações não adicionadas

Mostra o que mudou nos arquivos que ainda não foram para o stage:

git diff

Ver alterações já adicionadas

Mostra o que mudou nos arquivos que já estão no stage (após git add):

git diff --staged

Comparar dois commits

Compara as diferenças entre dois commits usando o hash deles:

git diff abc1234 def5678

Comparar com a branch main

Veja o que mudou na sua branch em relação à main:

git diff main

Exemplo completo

git diff
git diff --staged
git diff main
💡 Dica: Linhas em verde (+) são adições e linhas em vermelho (-) são remoções no diff.
