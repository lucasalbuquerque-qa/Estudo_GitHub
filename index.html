📖 Guia de Referência

Estudo GitHub

Consulte os principais comandos do Git organizados por categoria. Clique em qualquer comando para ver a explicação completa.

🗂️
Iniciar Projeto 4 comandos
git init Iniciar repositório local → git clone Clonar repositório remoto → git config Configurar usuário e e-mail → git ignore Ignorar arquivos no repositório →
🔍
Status e Histórico 3 comandos
git status Ver arquivos modificados → git log Ver histórico de commits → git diff Ver diferenças nos arquivos →
💾
Salvar Alterações 4 comandos
git add Adicionar arquivos ao stage → git commit Salvar alterações no histórico → git push Enviar commits ao GitHub → git stash Guardar alterações temporárias →
🌿
Branch (Ramos) 3 comandos
git branch Criar e listar branches → git checkout Mudar de branch → git merge Mesclar branches →
↩️
Desfazer / Corrigir 3 comandos
git revert Desfazer commit com segurança → git reset Voltar para commit anterior → git stash Guardar alterações temporárias →
🔄
Atualizar / Mesclar 3 comandos
git pull Baixar e aplicar atualizações → git fetch Baixar sem aplicar → git rebase Reorganizar histórico linear →
