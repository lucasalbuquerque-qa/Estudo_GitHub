git config Iniciar Projeto

O comando git config define as configurações do Git, como nome de usuário e e-mail. Essas informações aparecem em todos os commits que você fizer.

Quando usar

Use git config na primeira vez que instalar o Git, ou quando quiser usar uma conta diferente em um projeto específico.

Configurar nome globalmente

Define o nome para todos os projetos do computador:

git config --global user.name "Seu Nome"

Configurar e-mail globalmente

Define o e-mail para todos os projetos do computador:

git config --global user.email "seuemail@exemplo.com"

Configurar só para um projeto

Para usar uma conta diferente apenas no projeto atual, remova o --global:

git config user.name "Outro Nome"
git config user.email "outroemail@exemplo.com"

Verificar as configurações

Para ver todas as configurações ativas:

git config --list

Exemplo completo

git config --global user.name "Lucas"
git config --global user.email "lucas@exemplo.com"
git config --list
💡 Dica: Use --global para configurar uma vez e valer para tudo. Sem ele, a config vale só para o repositório atual.
