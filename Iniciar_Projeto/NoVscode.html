Primeiros comandos Git para iniciar um projeto no VS Code

1. Iniciar um repositório Git

Inicializa um repositório Git na pasta do projeto.

git init

2. Verificar status do projeto

Mostra quais arquivos foram modificados ou ainda não foram adicionados.

git status

3. Adicionar arquivos ao controle do Git

Adiciona todos os arquivos do projeto para serem salvos no Git.

git add .

4. Criar o primeiro commit

Salva as alterações no histórico do projeto.

git commit -m "Primeiro commit do projeto"

5. Conectar com o repositório do GitHub

Adiciona o repositório remoto.

git remote add origin https://github.com/usuario/repositorio.git

6. Enviar o projeto para o GitHub

Envia o projeto local para o repositório remoto.

git push -u origin main
