git init Iniciar Projeto

O comando git init cria um novo repositório Git vazio na pasta atual. É o primeiro passo para começar a versionar um projeto do zero.

Quando usar

Use git init quando você tem um projeto local que ainda não está sendo versionado pelo Git e quer começar a controlar as alterações.

Abrir a pasta do projeto no terminal

No terminal do VS Code, navegue até a pasta do seu projeto:

cd meu-projeto

Inicializar o repositório

Execute o comando para criar o repositório Git:

git init

Isso cria uma pasta oculta .git dentro do projeto, que é onde o Git guarda todo o histórico.

Verificar se funcionou

Confirme que o repositório foi criado:

git status

Definir a branch principal

Por padrão o Git cria a branch master. Para usar main (padrão atual):

git branch -M main

Exemplo completo

cd meu-projeto
git init
git branch -M main
git status
💡 Dica: Após o git init, o próximo passo é configurar seu usuário com git config e depois adicionar os arquivos com git add.
