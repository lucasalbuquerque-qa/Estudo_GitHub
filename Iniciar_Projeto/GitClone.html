git clone Iniciar Projeto

O comando git clone copia um repositório remoto do GitHub para o seu computador, criando uma pasta com todos os arquivos e o histórico completo.

Quando usar

Use git clone quando quiser baixar um projeto existente do GitHub para trabalhar localmente.

Copiar a URL do repositório

No GitHub, clique no botão Code e copie o link HTTPS do repositório.

Clonar o repositório

No terminal do VS Code execute:

git clone https://github.com/usuario/repositorio.git

Entrar na pasta criada

Após o clone, entre na pasta do projeto:

cd repositorio

Verificar o status

Confirme que o projeto foi clonado corretamente:

git status

Exemplo completo

git clone https://github.com/usuario/meu-projeto.git
cd meu-projeto
git status
💡 Dica: Se tiver mais de uma conta no GitHub, coloque o usuário na URL: https://usuario@github.com/usuario/repo.git
