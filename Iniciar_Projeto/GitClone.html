Iniciando um projeto com Git Clone

O comando git clone é utilizado para copiar um repositório do GitHub para o seu computador.

1. Copiar a URL do repositório no GitHub

No GitHub, clique no botão Code e copie o link do repositório.

2. Clonar o repositório

No terminal do VS Code execute:

git clone https://github.com/usuario/repositorio.git

3. Entrar na pasta do projeto

Depois que o repositório for baixado, entre na pasta criada:

cd repositorio

4. Verificar o status do projeto

Você pode verificar o estado dos arquivos com:

git status

Exemplo completo


git clone https://github.com/usuario/meu-projeto.git
cd meu-projeto
git status
